# 🌸 Vientres Proféticos

Plataforma digital ministerial desarrollada para la gestión de contenido espiritual, publicaciones, y almacenamiento multimedia.

---

## 🚀 Tecnologías utilizadas

- ⚛️ React
- 🟢 Node.js
- 🗄️ Supabase (Base de datos)
- 🖼️ Supabase Storage (Imágenes y multimedia)
- 🎨 CSS / Tailwind (Estilos)

---

## 📦 Instalación y ejecución

- npm run dev     

###  Instalar dependencias
 - npm install
